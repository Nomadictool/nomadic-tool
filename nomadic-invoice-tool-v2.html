Quick Start Guide

Step-by-step instructions for this tool

  1. Ask your Admin to unlock admin mode (lock icon, top-right).
  2. Admin clicks + New Incident (or Edit Incident on first launch).
  3. Fill in the Incident Name, E-Number, agency, billing period start date, and copy rates.
  4. Add staff names who will be working the fire.
  5. Click Start Incident →. The tool is now ready for orders.
  1. Whenever the tool loads and there are 2 or more active incidents, a fire selection screen will appear.
  2. Pick your incident from the dropdown and click Continue.
  3. You will only see data for your fire until you reload the page or switch incidents.
  4. To switch fires, reload the page and pick a different one, or ask Admin to use the Incident Manager.
  1. Go to the Daily Log tab. The date defaults to today.
  2. Click + Add Order to open the order form.
  3. Fill in Department, Job Description, and all applicable copy types (Black & White, Color, Plot, Legal, Custom).
  4. Click Save Order. The order appears in the daily log.
  5. Use the arrow buttons (< >) or the calendar icon to navigate between days.
  6. Helicopter / Communications units: this section is hidden — your unit bills a flat day rate with no copy orders. Just use ✓ Approve Day (see step 4) for each day worked.
  1. When a day's orders are complete (or, for Helicopter/Comms units, at the end of each day worked), click ✓ Approve Day in the Daily Log.
  2. Approved days are locked with a green badge. Editing requires a confirmation prompt.
  3. Only approved days (and days with orders) count toward the Incident Summary, Season Overview, and CSV export totals — so don't skip this step, even on a slow day with zero copy jobs.
  4. The Days panel (calendar icon) shows all dates with their Approved / Pending status.
  1. From the Daily Log, select the date you want to invoice, then click Preview Invoice → (or go to the Invoice tab directly).
  2. The Invoice tab shows that day's invoice and updates the page title to the correct file name (Date_E#_FireName_Daily-Invoice).
  3. Click 🖨 Print Invoice, then in the print dialog (Ctrl+P or Cmd+P) select Save as PDF.
  4. The PDF file name will match the page title — no renaming needed.
  1. Go to the Shift Log tab.
  2. Under Shift Notes, enter your name, date, and a handoff message. Click Post Note.
  3. All team members see the running note feed — use this to communicate complex orders and work in progress between shifts.
  4. Use Morning Checklist and Evening Checklist to track daily inventory and equipment checks.
  5. Check off items as you go. Click Reset at the start of each shift to clear the previous session's checks.
  1. Admin unlocks, then goes to the Rates tab.
  2. Click Export Backup to download a JSON file with all incident data and rates.
  3. Store the backup in a safe location (Google Drive, USB).
  4. To restore, click Restore Backup and select the JSON file. This replaces all current data.
  1. Admin unlocks, then clicks Season in the top nav.
  2. View a summary of all active incidents: total orders, revenue, approved days.
  3. Use this to replace your tracking spreadsheet at a glance.
